# word-counter1

a.
项目描述：
这是一个命令行文本计数统计程序。能正确统计导入的纯英文txt文本中的字符数，单词数，句子数，代码行、空行、注释行，并提供相应命令接口。
项目用法：
命令模式： wc.exe [参数] [文件名]
例：wc.exe -c file.txt 统计字符数
    wc.exe -w file.txt 统计单词数
    wc.exe -l file.txt 统计行数
    wc.exe -a file.txt 统计代码行、空行、注释行数
